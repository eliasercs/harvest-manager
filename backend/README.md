<h3 align="center">Harvest Manager - Backend</h3>

## Getting Started

### Requirements

* Node JS
* Build the environment variables file:

```
PORT=
MONGODB_CNN=
```

### Installation

Install all dependencies of Node JS.

~~~bash
> npm install
~~~

Run the project in mode of develop.
~~~bash
> npm start
~~~
